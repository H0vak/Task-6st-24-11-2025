Greedy Algorithm for Fuel Cost Optimization (C++)
Цей проєкт реалізує жадібний алгоритм для знаходження мінімальної вартості подорожі між містами з урахуванням обмеженого об'єму паливного бака. Робота демонструє вміння розв'язувати задачі на мінімізацію витрат та ефективне керування ресурсами.

Tехнології:
C++.

Алгоритмічна стратегія (Greedy Approach): Програма приймає рішення про дозаправку в кожному місті, базуючись на поточній ціні пального (яка дорівнює номеру міста) та залишку в баку.

Бібліотека <windows.h>: Для забезпечення коректної роботи з кириличними шрифтами в інтерфейсі консолі.

Що робить ця програма:
Розраховує стратегію заправки: Програма визначає оптимальну кількість пального для покупки в кожному місті, щоб доїхати до кінцевого пункту з мінімальними грошовими витратами.

Враховує обмеження об'єму: Алгоритм стежить, щоб кількість пального в баку ніколи не перевищувала його максимальну місткість V.

Динамічне ціноутворення: Моделюється ситуація, де в кожному наступному місті пальне стає дорожчим, що змушує алгоритм заправлятися максимально вигідно на початкових етапах.

Оптимізація ресурсів: Використовує просту логіку порівняння залишку пального з відстанню, що залишилася, для прийняття рішення про покупку 1 одиниці пального за ціною поточного міста.
